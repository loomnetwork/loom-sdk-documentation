OSX에서 Loom 설치 및 설정하기

## 설치하기

```bash
curl https://raw.githubusercontent.com/loomnetwork/loom-sdk-documentation/master/scripts/get_loom.sh | bash
chmod +x loom

./loom init
```

## 블록체인 실행하기

    ./loom run
